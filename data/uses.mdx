Dengan kondisi awal processor Intel® Core™ i3 Gen4 + RAM DDR3L 2GB onboard cukup frustasi untuk menjalankan Sistem Operasi [KDE Neon](https://neon.kde.org/) yang saya gunakan saat itu. Dengan menambah RAM 8GB dan menggunakan [Adata SU800 SSD 256GB](https://www.adata.com/us/specification/410) saya rasa sudah sangat jauh lebih baik dengan performa yang diberikan untuk kebutuhan menulis dan meng-compile kode.

### Misc

- Earbud menggunakan [Moondrop Nameless](https://www.moondroplab.com/) yang karakter suaranya cenderung _mid forward_, tentu saja irbat satu ini mampu membuat kuping saya _eargasm_ ketika mendengar suaranya mbak [Aimer](https://www.aimer-web.jp/). 
- Mouse pad menggunakan [Rexus Kvlar T5](https://rexus.id/produk/rexus-mousepad-gaming-t5/) yang sebenarnya biasa-biasa aja.
- Monitor Holder menggunakan stand holder [Robot RT-US05](https://tokopedia.link/e4DaHshaokb) yang sebenarnya dipergunakan untuk Smartphone dan _surprsingly_ ini cukup kokoh untuk menahan beban berat si LCD.

Terlalu banyak perangkat lunak yang berusaha memenuhi kebutuhan saya, tapi percayalah berikut merupakan beberapa perangkat lunak yang selalu saya gunakan, ~~kebanyakan~~ diantaranya merupakan open source software!

GNU/Linux

Sistem operasi yang saya gunakan saat ini adalah [Arch](https://archlinux.org/) dan menggunakan [i3wm](https://i3wm.org/) sebagai Window Manager. Menggunakan distro Arch karena alasan terdapatnya [AUR package](https://aur.archlinux.org/) yang memberikan kebebasan dalam memasang aplikasi.

**Dotfiles?** tentu saja saya menyimpan setiap baris konfigurasi yang saya terapkan dan ditaruh kedalam satu [repositori](https://github.com/opxop/dotfiles/).

[NVIM!](https://neovim.io/)

Nvim atau Neovim adalah penyunting kode utama saya saat ini setelah dibuat frustasi oleh [VSCode](https://code.visualstudio.com/) dengan RAM resource yang tidak lagi masuk akal untuk sekedar menulis kode html—kamu tau Electron App tidak RAM friendly. Kendalanya saya sering kali bolak-balik [cheatsheet](https://vim.rtorr.com/) karena belum terbiasa dengan keybind yang begitu detailnya setelah saya membuka [dokumentasi resminya](https://neovim.io/doc/user/).

### Misc

- Terminal emulator menggunakan [Kitty](https://sw.kovidgoyal.net/kitty/) karena mendukung multiple tab dan [st](https://st.suckless.org/) untuk merasakan suckless program.
- Peramban utama menggunakan [Firefox](https://www.mozilla.org/en-US/firefox/developer/), selain itu menggunakan [Brave](https://brave.com/) untuk beberapa alasan tertentu.
- Note-taking, todo list, dsb menggunakan [Notion](https://www.notion.so/) terlepas dari saya seorang yang produktif atau tidak.
- Password manager menggunakan [Bitwarden](https://bitwarden.com/) dan [Authy](https://authy.com/) untuk mengelola 2FA.
- Bookmark manager menggunakan [Raindrop](https://raindrop.io/).
- Membuat antarmuka dan desain menggunakan [Figma](https://www.figma.com/).
- Untuk mendengarkan lagu saya menggunakan [Spotify](https://spotify.com/) yang memiliki total kalatog lagu terbanyak (katanya), karena beberapa lagu dalam playlist saya pun hanya tersedia di platform ini.
